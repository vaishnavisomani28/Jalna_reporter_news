## ğŸ’¡ **Tips & Tricks**

### **For Images:**
1. **Upload vs URL:**
   - Upload: Better for permanent storage
   - URL: Faster, but link might break if source deletes image

2. **Image Placement:**
   - Upload images where you want them in content
   - Use editor toolbar to format around images

3. **Multiple Images:**
   - Upload as many images as needed
   - Each image uploads separately

### **For Links:**
1. **YouTube Links:**
   - Copy full YouTube URL
   - Paste in URL field
   - Add descriptive title

2. **Link Preview:**
   - Links show in blue box before adding
   - Can remove links before adding to content

3. **Link Format:**
   - Always starts with `https://` or `http://`
   - Must be valid URL format

## â“ **Common Questions**

**Q: Can I upload multiple images?**
A: Yes! Upload as many as you need, one at a time.

**Q: Where are uploaded images stored?**
A: In `/public/uploads/` folder on your server.

**Q: Can I delete uploaded images?**
A: Yes, from the server's uploads folder, or re-upload to replace.

**Q: Do YouTube links embed videos?**
A: Links are added as clickable links. Videos open on YouTube.

**Q: Can I add links without the "Add External Links" section?**
A: Yes! Use the link button in the editor toolbar.

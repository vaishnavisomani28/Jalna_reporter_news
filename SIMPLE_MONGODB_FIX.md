## Aap Yahan Ho: Database Users Page

## Ab Kya Karna Hai (3 Simple Steps):

### Step 1: IP Access List Pe Click Karo
1. Left sidebar mein **"NETWORK ACCESS"** section dekho
2. Uske andar **"IP Access List"** pe **CLICK** karo

### Step 2: Add IP Address Button Click Karo
1. Jab "IP Access List" page open hoga
2. Top right mein **"+ ADD IP ADDRESS"** ya **"Add IP Address"** button dikhega (green/blue button)
3. Uspe **CLICK** karo

### Step 3: Allow Access from Anywhere
1. Ek popup/modal box open hoga
2. **"ALLOW ACCESS FROM ANYWHERE"** button click karo
   - Ya phir manually `0.0.0.0/0` type karo
3. **"Confirm"** button click karo
4. **Done!** 1-2 minutes wait karo
